# AI Chatbot 

## Overview
AI Chatbot is a web-based application developed using Python and Flask. The chatbot allows users to interact through a simple interface and receive real-time responses. This project is designed for learning chatbot development and web application integration.
---
## Features
- Interactive chatbot interface
- Real-time responses
- User-friendly design
- Built using Python and Flask
- Simple and responsive layout
- Easy to understand project structure
---
## Technologies Used

### Backend
- Python
- Flask
### Frontend
- HTML
- CSS
- JavaScript
---
## Installation
### 1. Clone the repository
```bash
git clone https://github.com/anua41066-eng/Ai_chatbot.git
```
### 2. Open the project folder
```bash
cd Ai_chatbot
```

### 3. Install required packages
```bash
pip install -r requirements.txt
```
### 4. Run the application
```bash
python app.py
```
---
## Usage
1. Run the Flask application
2. Open your browser
3. Visit:
```bash
http://127.0.0.1:5000
```
4. Start chatting with the AI chatbot
