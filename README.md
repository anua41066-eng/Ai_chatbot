# AI Chatbot Project

## About
This is an AI Chatbot project developed using Python and Flask. The chatbot can interact with users and provide responses based on user input.

## Features
- User-friendly interface
- Interactive chatbot responses
- Built using Python and Flask
- Simple and responsive design

## Technologies Used
- Python
- Flask
- HTML
- CSS
- JavaScript

## Installation
1. Clone the repository:
git clone <your-github-link>

2. Open the project folder:
cd project-name

3. Install required packages:
pip install -r requirements.txt

4. Run the application:
python app.py

## Usage
Open the application in your browser and start chatting with the chatbot.
